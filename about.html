About This Project

This prototype showcases the implementation and integration of RESTful APIs to explore how APIs can help users retrieve rich-media and associated metadata. Future development will include full integration of video and audio playback without leaving the application, along with expanded metadata fields.

ChatGPT-5.2 assisted with development of the API integration script and responsive styling. It generated a JavaScript Fetch API template for returning data objects and metadata. All concepts, written content, and final design decisions were created by the author.

Poly Haven

Poly Haven is an open 3D asset repository widely used by game developers and digital artists. It provides high-resolution textures, models, and environment assets.

Wiki Commons

This tool uses the MediaWiki Action API to access data from Wikimedia Commons.
