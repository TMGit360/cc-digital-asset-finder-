About This Project

This prototype demonstrates how RESTful APIs can support discovery of openly licensed media and associated metadata through a clean web interface.

The application uses the Wikimedia Commons API to retrieve media records and present structured metadata fields.

ChatGPT-5.2 assisted with development of the API integration script and responsive styling. All concepts, written content, and final design decisions were created by the author.

Wikimedia Commons

This tool uses the MediaWiki Action API to access data from Wikimedia Commons.
